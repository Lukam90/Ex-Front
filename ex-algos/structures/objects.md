## Objects

An object is an unordered collection of key-value pairs.

The key must either be a string or symbol data type where as the value can be of any data type.

An object is not iterable.

insert - O(1)

remove - O(1)

access - O(1)

search - O(n)

Objects.keys() - O(n)

Objects.values() - O(n)

Objects.entries() - O(n)
